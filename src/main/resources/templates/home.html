Cluster is Unresponsive

Unable to retrieve cluster node status. This can be for several reasons:

  • There are not enough available nodes in the cluster, check the DB Console.
  • You are using a CockroachDB self-hosted cluster and have not started any nodes. Run ./cluster-admin start-all.
  • You are using a CockroachDB Cloud or Secure cluster and have no valid access token. Run ./cluster-admin login to authenticate.
  • You are using the wrong credentials (connecting to a secure cluster in insecure mode, etc). Check config/settings.sh.
